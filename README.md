NaijaSchoolFees Planner is a free, privacy-friendly school fees savings calculator built for Nigerian parents.  
It helps families calculate how much they need to save monthly to comfortably pay school fees before the next term.

---

Features

- Monthly school fees savings calculator  
- Designed for Primary and Secondary schools in Nigeria  
- Light and Dark mode  
- Works offline (static site)  
- No login, no tracking, no data storage  
- Mobile-friendly and fast
